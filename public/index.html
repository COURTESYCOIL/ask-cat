Getting Started: Make Your Own Ask Cat Bot

Want to create your own Ask Cat bot and run it in your terminal? Follow these simple steps:

1. Clone the Repository

First, clone the Ask Cat repository to your local machine:

git clone https://github.com/your-username/ask-cat.git
cd ask-cat

2. Install Dependencies

Install the necessary Node.js dependencies:

npm install

3. Configure Environment Variables

Create a .env file in the root of your project and add your Discord bot token and other necessary configurations. Refer to the .env.example file for required variables.

DISCORD_BOT_TOKEN=YOUR_BOT_TOKEN
# Other environment variables as needed

4. Run Your Bot

You can now run your Ask Cat bot directly from your terminal:

npm start

Your bot should now be online and ready to interact!

/api/progress

GET

Retrieves the interaction count and unlocked achievements for a user.

Parameters:

  • userId (query, required): The ID of the user.
  • x-bot-api-key (header, optional): API key for bot authentication.
  • auth-token (cookie, optional): JWT token for website authentication.

POST

Updates the interaction count and unlocked achievements for a user.

Parameters:

  • userId (query, required): The ID of the user.
  • x-bot-api-key (header, optional): API key for bot authentication.
  • auth-token (cookie, optional): JWT token for website authentication.

Request Body:

{
    "interactionCount": 10,
    "unlockedAchievements": {
        "first_login": true
    }
}

/api/me

GET

Retrieves information about the currently authenticated user.

Parameters:

  • auth-token (cookie, required): JWT token for website authentication.

/api/auth/callback

GET

Handles the Discord OAuth2 callback.

Parameters:

  • code (query, required): The authorization code from Discord.

/api/interactions

POST

Receives and processes interactions from Discord.

Parameters:

  • x-signature-ed25519 (header, required): The signature of the request.
  • x-signature-timestamp (header, required): The timestamp of the request.

Request Body:

{
    "id": "123456789012345678",
    "application_id": "987654321098765432",
    "type": 1,
    "token": "a_discord_interaction_token",
    "version": 1
}
